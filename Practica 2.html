1. Vecinos Próximos¶

A continuación vamos a ejecutar un código que prueba la clasificación mediante vecinos próximos para 1,3,5,11,21 y 51 vecinos. Usaremos los ficheros "example1.data", "example2.data", "example3.data", "example4.data" y "wdbc.data".

Cabe destacar que solo lo probaremos con validación cruzada con 10 particiones, ya que es el tipo de partición más normal y si no tendríamos que hacer 90 ejecuciones para mostrar todos los casos.

Primero se probará con normalización y después sin normalizar.

%%python3
from Datos import Datos
import EstrategiaParticionado
import Clasificador
import numpy as np

print("EJECUCION DEL ALGORITMO DE VECINOS PROXIMOS\n\n")
vecinos=[1,3,5,11,21,51]
ficheros=["example1","example2","example3","example4", "wdbc"]
for f in ficheros:
    for v in vecinos:
        print("EJECUCION DEL FICHERO "+f+" CON "+str(v)+" VECINOS:\n")
        data = 'ConjuntosDatos/'+f+'.data'
        dataset=Datos(data)
        estrategia= EstrategiaParticionado.ValidacionCruzada(10)
        clasificador=Clasificador.ClasificadorVecinosProximos(v,True)
        errores=clasificador.validacion(estrategia,dataset,clasificador,laplace=1, seed=None)
        print("Probabilidad de error: ",errores)
        media = np.mean(errores)
        d_tipica = np.std(np.transpose(np.array(errores)), axis=0)
        print("Promedio de error: ", media)
        print("Desviacion Tipica: ", d_tipica)
        print()
